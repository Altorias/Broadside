# 怒海孤帆 · Broadside

致敬《Seven Seas Deluxe》(2001) 的回合制海战游戏，使用 **React + TypeScript + Vite** 构建。纯前端、无后端，关卡**程序随机生成**，每一局都不重样。

## 快速开始

```bash
cd broadside
npm install
npm run dev        # 开发服务器 http://localhost:5173
npm run build      # 生产构建到 dist/（纯静态，可部署到任意静态托管）
npm run preview    # 本地预览生产构建
npm test           # 运行全部测试（引擎 + 生成器 + 模拟 + UI）
```

要求 Node.js 18+。

## 玩法

你驾驶一艘孤帆，被海盗船群围困。每回合**二选一**：

- **移动一格**（8 向）——移动方向决定船头朝向
- **舷侧齐射**——向船身左右两侧同时开炮，射程 3 格

海盗船每回合朝你逼近一步（快速海盗走两步），**撞上你就损失 1 条命**。炮打不完就**借刀杀人**：

| 消灭方式 | 得分 |
|---|---|
| 舷炮击沉 | 100 |
| 引诱撞上岛屿 / 礁石 / 残骸 | 250 |
| 引诱两船互撞（留下残骸障碍！） | 200 × 2 |
| 引诱坠入漩涡 | 250 |

你自己驶入漩涡会被**随机传送**到安全水域——绝境逃生用，落点风险自担。清光所有海盗船即过关；关卡无限、难度渐增（敌船更多、快速海盗混入）。初始 3 条命，每 5000 分奖 1 命。

### 操作

| 输入 | 操作 |
|---|---|
| 点击相邻格 | 移动（悬停可预览移动后的射界） |
| 点击自己的船 | 开炮 |
| `WASD` / 方向键 | 四向移动 |
| `Q` `E` `Z` `C` | 斜向移动 |
| `空格` / `F` | 开炮 |
| 动画中点击 / 按键 | 跳过动画 |

界面**常显你当前的左右舷射界**（能命中的格红色高亮）——朝向就是一切。

## 架构要点

**引擎与 UI 严格分离。** `resolveTurn(state, action)` 是纯函数，返回新状态 + **事件列表**；UI 按事件顺序"分拍"播放动画。逻辑可在 node 中毫秒级全量测试，动画与规则永不纠缠。

**微步制回合结算。** 快速海盗"走两步"不是特殊路径逻辑，而是把敌方阶段拆成两个微步：微步 1 全体动一步并结算冲突，微步 2 仅快速海盗重新瞄准再动。每格进入都接受完整结算，无跳格特判。

**全链路种子化。** `runSeed → levelSeed → rngState`，随机数生成器状态就存在 GameState 里——同种子同操作序列**完全可复现**，这是第二阶段肉鸽模式 daily seed 的地基。

**约束下的随机生成。** 随机撒障碍后校验四条公平性约束（出生点立足空间 / 敌船距离 / 水域连通率 ≥80% / 敌船与玩家连通），不满足重掷，失败降档，100 次全败落到确定性保底关——随机但永不出坏局。

**一个有趣的数学事实**：对同一目标贪心移动的两艘船，数学上不可能"对穿"（sign 向量无法互指）。规则里的对穿判定是为未来 AI 变体预留的兜底，测试用穷举固化了这个不可能性。

**8 连通语义一致。** 船可斜向穿过两块角相接的障碍之间的缝隙——移动、连通性校验、（炮线本就是直线）三处语义统一。

## 肉鸽模式（第二阶段，规划中）

引擎已参数化预留：`PlayerStats`（射程/命数/奖命门槛）与 `LevelConfig`（棋盘/障碍/敌配置）均为只读注入。届时叠加：连续关卡 run、关间三选一强化（射程+1 / 残骸弹 / 额外命…）、永久死亡、每日种子挑战。
